# Chadow Battle Limit

Мод для клиента **Мир танков** (WG), который **блокирует кнопку «В бой»** после заданного числа завершённых боёв за текущую сессию.

## Как работает

1. Вы выбираете лимит боёв (например, 5).
2. После каждого завершённого боя счётчик увеличивается.
3. Когда сыграно столько боёв, сколько выбрано, кнопка «В бой» становится неактивной и нажатие блокируется.
4. Счётчик и лимит сохраняются в `mods/configs/chadow.battle_limit.json` между перезапусками клиента.

Мод **не автоматизирует бой** и не влияет на стрельбу — только ограничивает возможность снова встать в очередь.

## Установка

Версию клиента смотрите в `version.xml` в корне игры. Для **Tanki** (`C:\Games\Tanki`) сейчас это **1.43.0.0**.

### Вариант A: `.mtmod` (рекомендуется)

1. Соберите мод из папки `wotmods` (нужен Python 2.7 и 7-Zip), либо скопируйте файлы вручную (вариант B).
2. Положите `chadow.battle-limit_X.Y.Z.mtmod` сюда:
   ```
   C:\Games\Tanki\mods\1.43.0.0\
   ```
   (при обновлении игры папка версии может смениться — смотрите `mods\` и `version.xml`)
3. Скопируйте конфиг:
   ```
   C:\Games\Tanki\mods\configs\chadow.battle_limit.json
   ```
   Папку `mods\configs` создайте, если её нет.
4. Запустите игру. В `C:\Games\Tanki\python.log` должна появиться строка `[chadow.battle_limit] loaded`.

### Вариант B: `res_mods`

Скопируйте содержимое `wotmods/res/` в:

```
C:\Games\Tanki\res_mods\1.43.0.0\res\
```

Скомпилируйте `.py` в `.pyc` (Python 2.7):

```bat
python -m compileall res\scripts\client\gui\mods
```

## Настройка лимита

### Кнопка в ангаре

Внизу справа отображается счётчик `0/5` (или `БАН` при полном запрете случайного боя). Нажмите — откроются настройки.

Лимит действует **только на случайный бой**. Рейтинг, клановые и другие режимы не блокируются.

### Файл конфигурации

`mods/configs/chadow.battle_limit.json`:

```json
{
  "enabled": true,
  "maxBattles": 5,
  "battlesPlayed": 0,
  "showNotifications": true,
  "randomOnly": true,
  "hardBlockRandom": false
}
```

| Поле | Описание |
|------|----------|
| `enabled` | Включить ограничение (`maxBattles` > 0) |
| `maxBattles` | Лимит случайных боёв (0 = без ограничения) |
| `battlesPlayed` | Сколько случайных боёв сыграно в сессии |
| `showNotifications` | Уведомления в ангаре |
| `randomOnly` | Ограничение только для случайного боя (рекомендуется `true`) |
| `hardBlockRandom` | Полный запрет случайного боя; другие режимы доступны |

### Горячие клавиши (в ангаре, опционально)

| Сочетание | Действие |
|-----------|----------|
| `Alt+Shift+1` … `Alt+Shift+9` | Лимит 1–9 боёв |
| `Alt+Shift+0` | Отключить лимит |
| `Alt+Shift+B` | Переключить пресеты: 1 → 3 → 5 → 10 → 15 → 20 → 30 → 50 → выкл |
| `Alt+Shift+R` | Сбросить счётчик сыгранных боёв |

Горячие клавиши дублируют настройки из окна по кнопке счётчика.

## Сборка `.mtmod`

Требования: [Python 2.7](https://www.python.org/downloads/release/python-2718/), [7-Zip](https://www.7-zip.org/).

```bat
build.bat -v 1.0.0
```

Скрипт компилирует Python, подставляет версию в `meta.xml` и упаковывает `chadow.battle-limit_1.0.0.mtmod`.

Подробнее о структуре Python-модов: [wotstat.info — настройка окружения](https://docs.wotstat.info/guide/first-steps/environment/python/).

## Совместимость

- Клиент **WG «Мир танков»** (проверяйте на своей версии).
- Ангар с кнопкой «В бой» в шапке (`LobbyHeader`).
- Для **АО «Айти Технологии» / «Мир танков» (RU)** классы UI могут отличаться — потребуется адаптация.
